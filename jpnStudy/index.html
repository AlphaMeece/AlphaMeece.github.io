How to Use

Selecting a Set

Initially, no set will be loaded, to load a set, click the three horizontal bars in the top left. Each set will be displayed with a light box followed by the name of the set. Clicking on the name of the set will expand it, showing each group of terms. You can click on a group(which will turn its name green) to select it and put it into the study tool. Alternatively, clicking the light box next to the set name will select the entire set(every group) and put it into the study tool.
